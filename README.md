# 💎 Diamond Price Analysis

## 📌 Project Overview
This project focuses on analyzing diamond prices based on different attributes such as carat, cut, color, and clarity. 
Interactive visualizations are created to understand price distribution and key factors affecting diamond prices.

## 🎯 Objective
To identify how different features of a diamond influence its price and to build meaningful visual insights.

## 🛠 Tools & Technologies Used
- Power BI
- Microsoft Excel / CSV Dataset
- Data Cleaning
- Data Visualization

## 📂 Dataset Features
- Carat – Weight of the diamond
- Cut – Quality of the cut
- Color – Diamond color grading
- Clarity – Purity level
- Price – Cost of the diamond
- Depth
